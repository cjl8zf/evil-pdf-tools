# evil-pdf-tools

## Summary

This package adds vim navigations key bindings to pdf-tools and also simulates the vim style of search for pdfs.

## Installing

You will need Emacs 24+, `make` and [Cask](https://github.com/cask/cask) to
build the project.

    cd evil-pdf-tools
    make && make install
